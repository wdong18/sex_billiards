# Billiards Game

A simple billiards game created with HTML, CSS, JavaScript, and the Matter.js physics engine.

## Features

- Realistic physics simulation using Matter.js
- Two-player gameplay
- Score tracking
- Pockets and ball collision detection
- Aiming mechanism with visual guide
- Reset functionality

## How to Play

1. Open `index.html` in your web browser.
2. Click and drag from the cue ball to set the direction and force.
3. Release to strike the ball.
4. Players take turns after each shot.
5. Score points by pocketing balls.
6. The first player to sink the 8-ball (black ball) after pocketing at least one of their assigned balls wins.

## Controls

- **Mouse Down**: Start aiming
- **Mouse Move**: Adjust aim direction and power
- **Mouse Up**: Strike the cue ball
- **Reset Button**: Start a new game

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Matter.js Physics Engine

## Setup

No installation required. Simply clone or download the repository and open `index.html` in a web browser.

```bash
git clone https://github.com/yourusername/billiards-game.git
cd billiards-game
```

Then open `index.html` in your browser.
